# さじグラム

レシピの大さじ・小さじ表記をグラムに変換するシンプルな Web アプリ。

🔗 **URL**: https://ichibeikatura.github.io/sazzzy/

---

## 使い方

1. **さじ種別** — 大さじ／小さじを選ぶ
2. **分量** — 整数・分子・分母のプルダウンを組み合わせて入力（例: 1と2/3）
3. **調味料** — プルダウンから選ぶ

結果はリアルタイムで表示されます。

---

## 調味料マスタ

| グループ | 大さじ1 | 小さじ1 | 調味料 |
|---|---|---|---|
| 15g（比重≒1.0） | 15g | 5g | 水・酒・酢 |
| 18g（比重≒1.2） | 18g | 6g | みりん・しょうゆ・塩 |
| 9g（比重≒0.6） | 9g | 3g | 小麦粉・片栗粉 |

### 調味料の追加方法

`index.html` 内の `ingredients` 配列に1行追加するだけです。

```js
const ingredients = [
  { name: "水",     group: 0 },
  { name: "砂糖",   group: 1 }, // ← こんな感じで追加
  // ...
];
```

`group` はグラム値に対応したインデックスです（0=15g / 1=18g / 2=9g）。
新しい比重の場合は `groups` 配列にグループを追加し、`group` に新インデックスを指定してください。

---

## 計算式

```
グラム = (整数 + 分子/分母) × さじ係数 × 大さじ1あたりのグラム数
```

- 大さじ係数 = 1（15ml）
- 小さじ係数 = 1/3（5ml）
- 結果は小数第2位で四捨五入、割り切れる場合は整数表示
